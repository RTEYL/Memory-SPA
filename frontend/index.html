Memory Game

Welcome to the Memory Game!

Instructions

To begin click the 'Play' button

    Once the game starts:

  • A random box will highlight

  • Click the correct highlighted box

  • The next box will highlight

  • Click the first box then click on the new box

  • The goal is to remember the order of each box as it's highlighted the longer you play the more difficult it will be to remember the correct order of boxes to click

You got Points!
